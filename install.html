Installation

To install the kit, download the .rbxl or .rbxm file from the releases page of the GitHub repository or download the roblox model directly from the toolbox in Roblox Studio.

For GitHub access to the model, go here. Note that some releases do not have a model available and is script-access only through Git, so please pick the right release.

Then, import your model into Roblox Studio. If the model has any extra objects, don't worry, I might have accidentally put something in. It should look somewhat like this:

In workspace, or where ever you put your main folder, there are sub folders that are named the game's services like so:

Drag those folders into their designated game services and then ungroup that specific folder (Ctrl + U)

The game should work as planned and if there is any issues, please contact me at .eychbee or in the issues tab.
