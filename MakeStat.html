Making A Stat

Introduction

One of the worst things about a button simulator kit is to make a stat. Oftentimes it takes many minutes just to even figure out how the kit even works. In HBS Kit, making stats is a piece of cake.

Stats, their orbs, and other attributes all rely on one script, InteractiveAlya.luau (2 if you count GUI). Within this script, the main functionality only consists of 75 lines, which is miniscule compared to other Button Simulators's orb functionality.

We are still working on how to make stats even easier to make, so if it's confusing, just bear with us.

Step 1: Open up the script and scroll all the way down.

Step 2: Copy the following code into the last empty line: 

                    
    local TestStatTable = {
        Tag = "TestStat",
        Boosts = {
            ["TestStat"] = "1000001e-5"
        },
        CostTag = "Money",
        RetuTag = "TestStat",
        Attributes = {
            Reset = {},
            SubFromCTag = false,
            BillboardFormat = "Cost CostTag = Return RetuTag"
        }
    }
